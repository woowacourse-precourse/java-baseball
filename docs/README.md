# 기능 목록

- [x] 입력한 수에 대한 결과 판단
- [x] 컴퓨터 숫자 선택(서로 다른 3자리의 수이면서 각 자리 숫자가 1~9 사이)
- [x] (게임 중) 사용자 숫자 입력 받기
    - [x] (예외처리) 각 자리가 1~9사이인 세자리 숫자가 아닐때
    - [x] (예외처리) 각 자리 중 서로 같은 수가 있을때
- [x] (게임 종료시) 사용자에게 숫자 1 또는 2 입력 받기
    - [x] 입력한 수에 대해서 1이면 재시작, 2면 게임종료
    - [x] (에외처리) 입력 받은 숫자가 1 또는 2가 아닐때
