
## 기능 구현 목록

+ #### 컴퓨터의 숫자 랜덤 생성 
  + 1~9까지 서로 다른 숫자 3개 랜덤 생성
+ #### 입력 기능 생성 
+ #### 숫자 3자리 입력 (입력 기능 사용)
  + 숫자 이외 or 중복된 값을 받는 경우 IllegalArgumentExcepetion
+ #### 컴퓨터의 3자리 와 입력 3자리 비교 후 출력 
  + 볼 / 스트라이크 / 낫싱 처리 기능
+ #### 재시작, 종료 여부 (입력 기능 사용)
  + 1 : (재시작) 새로시작 구현
  + 2 : (종료) 종료 
  + 1,2 이외 값 : IllegalArgumentExcepetion
+ #### 예외기능 처리 
  + IllegalArgumentExcepetion
