1. 1~9 사이의 숫자를 뽑아 중복 없이 3자리 수를 정한다.

2. 입력받은 값
    - 입력받은 값이 3자리가 아닐 경우 exception
    - 서로 다른 3개의 숫자가 아닐 경우 exception
    - 입력받은 값이 숫자가 아닐 경우 exception

3. 입력 값에 따른 출력 값
    - 같은 수가 전혀 없음 -> 낫싱
    - 같은 수가 다른자리 -> 볼
    - 같은 수가 같은자리 -> 스트라이크

4. 게임 결과
    - 3스트라이크가 아니면 계속 진행
    - 스트라이크인 경우 게임 종료

5. 게임 종료 시
    - 게임을 재시작 하려면 1입력
    - 게임을 완전히 종료하려면 2입력
    - 입력받은 값이 3자리가 아닐 경우 exception
    - 서로 다른 3개의 숫자가 아닐 경우 exception
    - 입력받은 값이 숫자가 아닐 경우 exception

3. 입력 값에 따른 출력 값

    - 같은 수가 같은자리 -> 스트라이크

    - 스트라이크인 경우 게임 종료

5. 게임 종료 시
    - 게임을 재시작 하려면 1입력
    - 게임을 완전히 종료하려면 2입력
