# ⚾ 기능 목록 ⚾

---
- [X] 게임 시작 문구 출력하기

- [X] 컴퓨터의 3가지 랜덤 숫자 생성 
  - [X] `컴퓨터 숫자(일급컬렉션 리스트)`
  - [X] 숫자의 생성
  - [X] 숫자 반환   
  

- [X]  사용자의 숫자 입력 받기
    - [X] `사용자 숫자(일급컬렉션 리스트)`
    - [X] 예외 처리  
   
   


- [X] 사용자 입력과 컴퓨터 숫자 비교하기
  - [X] 볼의 숫자 계산
  - [X] 스트라이크의 숫자 계산
  - [X] 낫싱인 경우 확인
  - [X] 결과 출력


- [X] 게임 종료 문구 출력하기
 

- [X] 게임 시작 여부 묻기
    - [X] 1인 경우 새로 시작하기
    - [X] 2인 경우 종료하기
