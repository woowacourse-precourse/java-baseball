## 구현 기능 목록

1. 메인 기능  (run)
   - 하위의 모든 기능을 실행하는 메서드 
   

2. 랜덤 숫자 생성하는 기능(createRandomNum)
   - 랜덤 숫자 생성
   

3. 숫자입력 (guessNum)
   - 사용자의 추측 숫자 입력
   

4. 숫자 맞는지 확인하는 기능 (checkNum)
   - 랜덤 숫자와 사용자 입력 숫자 비교  


5. 종료하거나 다시시작 (closeOrRestart)
   - 종료 or 다시시작
   

6. 입력 사항 예외 기능 (BaseBallException) 
   - 함수로 빼서 여러가지 예외 기능 메서드 구현
