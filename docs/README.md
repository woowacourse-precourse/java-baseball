## 기능 목록 작성

1. 정답 생성
- 임의의 난수 생성 (1~9 서로 다른 수, 3자리)
2. 게임 진행
- 사용자 숫자 입력
- 사용자 입력 값 오류 확인 (IllegalArgumentException 발생)
    - 입력받은 수가 3자리가 아닌 경우
    - 숫자가 아닌 경우
    - 서로 다른 3자리 수가 아닌 경우
- 입력받은 수와 정답과 비교
    - 숫자가 일치하는 값이 없는 경우 : 낫싱
    - 숫자가 일치하나 자리가 다른 경우 : 볼
    - 숫자와 자리가 일치하는 경우 : 스트라익
- 3스트라이크면 게임 종료, 2스트라이크 이하면 계속해서 게임 진행
3. 게임 재시작/종료
- 사용자 숫자 입력
- 사용자 입력 값 오류 확인 (IllegalArgumentException 발생)
    - 1,2 이외의 숫자 혹은 문자가 입력되는 경우
- 1 입력 시 게임 재시작
- 2 입력 시 게임 종료
