### baseball 기능 목록 

1. 숫자야구 게임 시작 하는 기능 [ O ]
2. 사용자가 숫자를 입력하는 기능 [ O ]
3. 사용자가 잘못된 값을 입력할 경우 에러처리 [ O ]
   1) 입력받은 숫자의 길이가 1보다 작거나 3보다 클 때
   2) 입력받은 숫자가 숫자가 아닐 때
   3) 숫자 중복되는지 확인
4. 컴퓨터가 숫자를 랜덤으로 선택하는 기능 [ O ]
5. 스트라이커를 판독하는 기능 [ O ]
6. 볼을 판독하는 기능 [ O ]
7. 낫싱을 판단하는 기능 [ O ]
8. 3스트라이크일 때 게임종료 기능 [ O ]
9. 게임이 종료되었을때 다시 할지, 종료 할지 선택하는 기능 [ O ]
