## 로직
```
1. 시작 문구 출력
2. Computer 생성
3. 서로 다른 값의 3자리수 입력
4. 컴퓨터와 플레이어 숫자 값 비교
5. 힌트 출력
6. "3스트라이크"면 7번으로 / 아니면 3번으로
7. 재시작 여부 확인
8. "1" 입력 시, 재시작 / "2" 입력 시, 종료
```
## 예외처리
```
사용자가 잘못된 값을 입력시 IllegalArgumentException 예외
   1. 3개의 숫자가 아닐 시
   2. 3개의 숫자가 서로 다른 숫자가 아닐 시
```
