## 기능구현 목록
___

### 컴퓨터
- 컴퓨터 숫자 생성기능
   - 각 자리수가 중복되지않는 세자리 자연수 생성

### 유저
- 유저가 숫자입력기능
  - 콘솔창에 유저가 세자리 숫자 입력
- 유저가 입력한 문자를 저장하는 기능


### 게임 시스템
- 스트라이크/볼 비교기능
  - 유저입력 자연수와 컴퓨터 숫자를 각 자리수 비교후 결과 출력
  - 3스트라이크 되기까지 반복
- 끝난 후 종료 or 재시작 선택
    
### 예외
- 유저
  - 입력값 길이가 3이 아닐경우
  - 입력값중 문자 혹은 0 인경우 (1 ~ 9까지)
  - 입력값중 중복된 문자가 포함되어있는 경우
- 게임 시스템
  - 1, 2이외의 값을 입력하는 경우
