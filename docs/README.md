## 🌼 워크플로우
1. 입력 받기
   - 입력 예외처리
     1. 3자리가 넘는 경우(`12345`)
     2. 동일한 숫자(`112`)
     3. 숫자가 아닌 문자가 들어오는 경우(`a23`)
     4. 아무것도 들어오지 않은 경우(` `)
     5. 사이 공백이 있는 경우(`1 2 3`)
   - 3자리 랜덤 난수 생성 (상대편)
2. 입력값 처리 및 결과 출력
    - 문자열 비교 후 List로 `[N-ball, N-strike]` 반환
    - 반환된 List를 통해 힌트 제공 또는 게임 종료
3. 게임 종료 시 재시작/종료 선택
   - 입력 예외처리
