## 기능 목록
1. 입력
- 게임 진행중 (User 값)
  - 3자리 
  - 1~9 사이의 숫자
  - 중복 여부 
- 게임 끝난후 (재시작 여부 판단)
  - 재시작 : 1
  - 게임 종료 : 2
- 입력값 공통사항
  - 사용자가 잘못된 값 입력시 오류 처리 후 게임 종료

2. 게임 진행
- 컴퓨터: 서로다른 3자리 랜덤 숫자(1~9) 조합 생성
- User와 Computer의 값 비교
  - 입력 받은 User 값과 랜덤 생성한 Computer 값 비교
    - 위치  값 
    -  O   O  스트라이크
    -  X   O  볼
    - 스트라이크/볼 존재x  낫싱 
  - 비교 결과로 스크라이크,볼 수 or 낫싱 출력
  - 3스트라이크 나올때까지 반복
    - 3스트라이크시 재시작 여부 확인
