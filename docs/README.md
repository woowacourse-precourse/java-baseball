# 구현 기능 목록

## Util
1. 랜덤 숫자 3개 생성 메소드
2. 3개 숫자 입력 메소드
3. 게임 후 재시작 의사 숫자 입력 메소드

## Game
1. 게임 시작 메소드
2. 게임 승리시 재시작 유무 메소드
3. 게임 재시작 및 혹은 종료 메소드

## Computer
1. 랜덤 숫자 3개 받기 메소드
2. 스트라이크 확인 메소드
3. 볼 확인 메소드
4. 게임 결과 처리 메소드

## Exception
1. 입력값 예외처리 메소드
    1. 1 ~ 9 숫자만 가능
    2. 중복 숫자 불가
    3. 3자리 숫자
    4. 재시작 및 종료 입력값 1, 2 외 불가
