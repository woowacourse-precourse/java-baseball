## 🚀 구현 기능 목록

- [X] 컴퓨터 3자리 수 생성
    - [X] 서로 다른 임의의 수 3개 생성
    - [X] 중복 값이 있는지 확인
    - [X] 1~9 사이의 숫자인지 확인

- [X] 유저 숫자 생성
    - [X] 유저한테 수 입력 받기
        - [X] 인풋 값이 중복되지 않았는지 확인
        - [X] 인풋 값의 길이가 조건에 맞는지 확인
        - [X] 인풋 값에 0이 포함되어 있는지 확인

- [X] 결과 계산
    - [X] 스트라이크 갯수 계산
    - [X] 볼 갯수 계산

- [X] 힌트 출력
    - [X] 볼/스트라이크/낫싱을 결과 따라 출력

- [X] 게임 시작 및 종료
    - [X] 숫자 야구 게임을 시작합니다. 출력
    - [X] "숫자를 입력해주세요 : " 출력
    - [X] 랜덤 숫자 만들기
    - [X] 유저에게 숫자 받기
    - [X] 게임 클리어 여부 확인
    - [X] 클리어 시 축하 메시지 출력
    - [X] 게임 재시작 또는 종료

- [X] 게임 반복
    - [X] "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요." 출력
    - [X] 유저한테 수 입력 받기
        - [X] 인풋 값이 1 또는 2가 아닐 경우 예외 처리
