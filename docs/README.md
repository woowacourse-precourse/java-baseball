# 📕기능 목록

### 숫자 생성 & 입력 <Settings>
- ✅ 컴퓨터가 임의의 숫자 3개 선택
- ✅ 사용자한테 숫자 입력받기
- ✅ 사용자가 입력한 수 리스트에 저장
- ✅ 사용자 입력 예외처리
    - ✅ 게임 중 입력한 숫자 예외 처리
    - ✅ 게임 종료 시 입력하는 숫자 예외 처리

### 숫자 판단 <Game>
- ✅ 스트라이크 판단
- ✅ 볼 판단
- ✅ 낫싱 판단

### 게임 진행
- ✅ 3스트라이크 될때까지 게임 진행
- ✅ 숫자 판단 결과 출력

### 게임 종료
- ✅ 게임 시작 문구 출력
- ✅ 게임 진행
  - ✅ 1이면 처음으로 돌아가서 재시작
  - ✅ 2이면 게임 종료
