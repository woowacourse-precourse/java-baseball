## 기능 구현 목록

### 1. domain
>##### Computer
>* 랜덤 난수를 통한 숫자 리스트 생성 기능
>* 랜덤 난수 숫자 리스트 삭제, 조회 기능
>
>##### User
>* 유저의 입력을 받는 기능
>* 유저 입력 숫자 리스트 저장, 삭제, 조회 기능
>
>##### GameRule
>* 현재 상태를 게임 룰에 반영된 결과 생성 기능
>* 게임 룰에 반영된 결과 조회 기능


### 2. filter
>##### UserInputFilter
>* 유저 입렵 예외 처리 기능
>* 유저 입력을 변환하여 리스트 반환 기능


### 3. view
>##### TotalViewImpl
> * 출력에 필요한 메시지 및 포멧 출력 기능


### 4. service
>##### GameService
> * 게임을 시작하는 기능
> * 게임을 재시작하는 기능
> * 게임을 종료하는 기능


### 5. Application
> * 어플리케이션 실행 로직
