# 기능 목록
## 입력 검증
- **게임 시작 시 세자리 수 입력**
    - 서로 다른 세자리 수를 입력했는가?
    - 각 자릿수는 1이상 9이하인가?
- **게임 종료 시 재시작 여부 입력**
    -  숫자 1또는 2를 입력했는가

## 컴퓨터의 수를 생성
- 1부터 9까지의 서로 다른 수로 이루어진 무작위 세자리 수를 생성한다.

## 입력받은 수를 판단
- 입력받은 수에 대해 스트라이크, 볼, 낫싱 여부를 계산
- 계산 결과에 따른 기능
    - 정답을 맞추지 못한 경우: 카운트 출력
    - 정답을 맞춘 경우: 정해진 문구 출력 및 '재시작 여부 확인'기능 실행

## 재시작 여부 확인
-  정해진 문구 출력
- 사용자에게 정수를 입력받음

## 예외처리
- 사용자가 규칙 외 값을 입력한 경우
