# 숫자 야구 게임

## 🗒 기능 구현 목록

---

### 1.난수 생성 (Computer)
- 상태방(컴퓨터)의 숫자를 랜덤으로 생성
- camp.nextstep.edu,missionutils를 사용하여 구현
- 예외처리: 
    - 중복 숫자인지 확인
    - 범위가 1~9인지 확인

### 2. 사용자 숫자 입력 (User)
- 입력 받은 숫자가 올바른 숫자인지 확인
- 예외처리:
    - 숫자가 아닌 경우 확인
    - 입력값의 길이가 3이 아닌 경우 확인
    - 서로 같은 수가 있을 경우 확인

### 3. 숫자 맞추기 (CompareNumber)
- 받아온 컴퓨터숫자와 사용자숫자의 리스트 인덱싱을 통하여 비교
- 비교된 스트라이크와 볼의 수대로 결과값 출력
- 숫자를 맞추면 true, 맞추지 못하면 false 반환
- 예외처리:
    - 각각의 경우의 수 모두 확인

### 5. 실행 (BaseBallGame)
- 구현한 클래스를 순서에 맞게 호출
- 필요한 파라미터와 반환값에 맞게 메서드 호출 
- 무한반복문과 종료조건 명시
- 사용자 숫자 입력
- 게임 재실행 여부 확인
- 예외처리:
  - 1,2 외에 다른것 입력시 확인
