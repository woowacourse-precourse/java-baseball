### 기능 목록

1. 랜덤한 3개의 숫자 생성. (중복되는 수 없음.)
2. 유저로 부터 숫자 입력 받음.
   1. 입력 받은 수 검증(중복, 0, 숫자 이외 입력)
3. 볼, 스트라이크 수 카운트
4. 볼 스트라이크 낫싱 정답 판별 및 결과 출력
5. 정답 맞춘 후 게임 재시작 할지 끝낼지 사용자 입력 받음.
   1. 입력 받은 수 검증

* 2, 5번 만들때 예외 처리해야 함.
  * 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료되어야 한다.
* 각 기능 만든 후 테스트 구현
