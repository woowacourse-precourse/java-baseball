# 숫자 야구 게임 클래스 구현
  
  - 숫자 야구를 진행하는 메서드
  
  - 사용자의 입력을 판별하는 메서드: 
  
        게임 진행 중, 게임 종료 시
  
  - 최초 컴퓨터의 숫자를 초기화하는 메서드
  
  - 사용자의 입력을 받는 메서드
  
  - 결과를 출력하는 메서드
  
  - 종료를 판별하는 메서드

  - 예외: 
  
         사용자가 3자리 숫자 이외를 입력했을 때
  
         사용자가 입력한 숫자중 같은 숫자가 있을 때
         
         사용자가 종료시 1과 2 이외의 숫자를 입력 했을 때
         
# 테스트 작성
  
  - 위의 3가지 예외 케이스에 대한 테스트 작성
  - 정상 작동 케이스에 대한 테스트 작성
