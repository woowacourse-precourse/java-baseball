> # 기능 구현 목록
> > 1. answer 숫자 3개 정하기
> > 2. Prompt 창 띄우기
> > 3. 사용자로부터 입력 받기
> > 4. 볼 개수 구하기
> > 5. 스트라이크 개수 구하기
> > 6. 프로그램 종료
> # 에러 처리 목록
> > 1. 사용자로부터 입력받은 문자열 길이가 3인지 확인
> > 2. 문자열이 모두 숫자인지 확인
> # 상수 목록 (예외 처리)
> > 1. 문자열 길이가 3이 아닌 경우 => LENGTH_NOT_3 = 1
> > 2. 문자열이 모두 숫자가 아닌 경우 => ALPHA_IN_USER = 2
