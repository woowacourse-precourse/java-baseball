# 기능 목록
## makeRandomNumbers
- 1에서 9까지 서로 다른 임의의 수 3개를 선택

## checkForSameNumbers
- 서로 다른 임의의 수인지 확인하기 위한 i 값을 반환

## splitInputToArray
- 입력받은 String값은 Array로 변환

## playBall
- 같은 숫자가 있는지 체크
  - 없으면 낫싱
  - 있으면 스트라이크
- 같은 숫자가 같은 자리에 있는지 체크
  - 없다면 그대로 스트라이크
  - 있다면 볼

## playGame
- 3스트라이크가 될 때까지 playBall 함수 실행
