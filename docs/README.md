# 기능목록

- 서로 다른 세개의 수 생성
  - 각 숫자의 범위는 1 ~ 9
  - 랜덤으로 생성

- 사용자 입력 검증
  - 입력된 값이 숫자
  - 입력으로 들어오는 숫자는 3자리
  - 각 숫자의 범위는 1 ~ 9
  - 잘못된 입력은 IllegalArgumentException

- 사용자 입력 분석
  - 각 자리별로 나눔

- 사용자 입력을 기반으로 게임 진행
  - 스트라이크 판별
  - 볼 판별
  - 낫싱 판별

- 게임 진행 결과에 따라 출력
  - 스트라이크
  - 볼
  - 스트라이크 + 볼
  - 낫싱
  - 정답
  - 정답 이외에는 다시 입력

- 게임 종료 여부 입력 검증
  - 숫자는 1 또는 2
  - 잘못된 입력은 IllegalArgumentException

- 입력에 따라 실행
  - 1 입력 시 재시작
  - 2 입력 시 종료
