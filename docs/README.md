- [X] 컴퓨터가 랜덤 숫자를 생성
- [X] 사용자가 숫자를 입력
    - [ ] 숫자 유효성 검사
        - [X] 입력값이 모두 숫자인가?
        - [X] 숫자의 범위가 1에서 9사이인가?
        - [X] 입력값의 길이가 3인가?
        - [ ] 중복된 숫자가 포함되는가?

    - [ ] 정답 메시지 출력
    - [ ] 재시작 혹은 종료 메시지 출력 후 사용자 입력

            - [ ] 입력값이 1 혹은 2인가?
        - [ ] 입력값에 따라 재시작 혹은 종료
