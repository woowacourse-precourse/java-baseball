# 공부한 내용 정리하기

### Pattern Matching(정규 표현식)
    Pattern p = Pattern.compile("(^[0-9]*$"));
    
    ^ : 문자열의 시작
    $ : 문자열의 종료
    [] : 문자의 집합이나 범위를 나타냄, 두 문자 사이에는 - 기호로 범위 나타냄
    * : 앞문자가 없을 수도 무수히 많을 수도 있음 -> 글자 수를 상관하지 않고 검사한다.
    즉, 0~9까지의 수를 글자 수 제한 없이 검사한다는 의미이다.

### 명명 규칙(클래스, 메소드, 변수)
    파스칼 표기법 :  모든 단어에서 첫 번째 문자는 대문자이며 나머지는 소문자이다.
    카멜 표기법 : 최초에 사용된 단어를 제외한 첫 번째 문자가 대문자이며 나머지는 소문자이다.
    클래스
        PascalCase사용
        명사형 사용
        영문+숫자+언더스코어
        ex) HelloWorld
    메소드
        camelCase 사용
        동사형 사용
        영문+숫자+언더스코어
        ex) searchData
    변수
        camelCase 사용
        명사형 사용
        영문+숫자+언더스코어
        ex) firstName
