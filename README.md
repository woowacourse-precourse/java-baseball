## ✏️ 기능 목록

- 자동
  - [X] 정답 (서로 다른 숫자 3개) 생성
  - [ ] 게임 기능
    - [ ] 3개의 숫자 모두 맞혔을 때 (3스트라이크)
      -  [ ] 1 입력시 재시작
      -  [ ] 2 입력시 종료
  - [X] 컴퓨터랑 사용자 입력 비교
    - [X] 맞힌 숫자가 없을 때 낫싱 출력
    - [X] 입력 숫자의 자리가 틀리고 정답에 포함일 때 볼
    - [X] 입력 숫자의 자리가 맞을 때 스트라이크

- 사용자
  - [X] 숫자 3개 입력
    - [X] 숫자 예외 체크
