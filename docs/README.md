## 🚀 기능 구현 목록
### 입력
- [x] 서로 다른 3자리 수를 입력받는다. (게임이 끝날때 까지)
   - *예외 (IllegalArgumentException)*
     - 3자리 수가 아닐 경우
     - 서로 다른 수가 아닐 경우
     - 숫자가 아닐경우
     - 아무것도 입력하지 않을 경우
- [x] 게임이 끝난 경우 재시작/종료 구분하는 1과 2 
  - *예외 (IllegalArgumentException)*
    - 1과 2가 아닌 수가 들어올 경우
    - 숫자가 아닐경우
    - 아무것도 입력하지 않을 경우
### 출력
- [x] 입력한 수에 대한 결과
  - 숫자 야구 시작
  - 스트라이크나 볼로 표시
  - 낫싱
  - 3스트라이크 게임 종료
