### 기능 명세서

- 가장 처음 숫자 뽑기
  - 랜덤
  - 1에서 9까지 임의의 수 3개
  - 게임 시작 문자 출력
- 플레이어에게 3 자리 수 입력받기
- 잘못된 입력일 때, exception 발생
  - 입력 값이 3자리가 아닌 경우
  - 중복된 숫자가 있는 경우
  - 숫자가 아닌 경우
  - `IllegalArgumentException`
- 입력받은 숫자 List형태로 변환
- 입력받은 숫자에 대한 힌트 출력
  - 같은 수 같은 자리 -> 스트라이크
  - 같은 수 다른 자리 -> 볼
  - 같은 수가 전혀 없음 -> 낫싱
- 게임 종료
  - 3개의 숫자를 모두 맞춘 경우
- 게임 재시작, 완전히 종료 여부 선택
  - 1 게임을 새로 시작
  - 2 게임을 완전히 종료
