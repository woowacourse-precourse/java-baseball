# ⚾ ️숫자야구 게임 기능 목록

### user
    [] 사용자 입력값이 (1, 9) 사이 정수인지 확인
    [] 각각 다른 수인지 확인
    [] 길이가 3인지 확인

### computer
    [] 랜덤 숫자 생성

### system
    [] 사용자 입력값과 컴퓨터 랜덤 숫자 비교 

### gameplay
    [] 게임 시작 문구 출력 {숫자야구 게임을 시작합니다.}
    [] 숫자 입력 안내 메시지 출력 {숫자를 입력해 주세요.}
    [] 결과 출력
    [] 게임 종료 문구 출력
    [] 게임 재시작 여부 물음 출력 {게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.}
        1. 재시작
        2. 게임 종료
