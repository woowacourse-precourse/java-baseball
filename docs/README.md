# 미션 - 숫자 야구
##   기능 구현 목록

---

### 컴퓨터 :
- 1~9에서 중복없이 숫자 3개를 정함

### 플레이어 :
  - 3자리 숫자를 입력받음
    - [에러처리] 숫자가 아닌 문자가 입력되는 경우
    - [에러처리] 0이나 동일한 숫자가 포함되는 경우
    - [에러처리] 3자리 숫자가 아닌 숫자가 입력되는 경우

### 게임 :
- 자리, 숫자가 모두 같으면 **스트라이크**
- 자리는 다르지만 숫자가 같으면 **볼**
- 숫자가 모두 다르면 **낫싱**
- 플레이어와 컴퓨터의 **숫자**, **자리**가 모두 같으면 **종료**
  - 재시작하려면 **1**을 입력
  - 종료하려면 **2**를 입력
  - [에러처리] 입력이 **1**이나 **2**가 아닌 경우
