### 게임 플레이어를 추상화한 클래스
    - [x] 각 플레이어가 공을 고르는 기능 (abstract)
    - [x] 고른 공을 가져오는 기능
    - [x] 고른 공을 초기화하는 기능
    - [x] 고른 공의 개수를 가져오는 기능

    - [x] 공을 고른 공 리스트에 추가하는 기능

        - [x] 중복된 공이 있는지 검증하는 기능
