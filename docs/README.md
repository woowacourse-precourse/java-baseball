# 베이스볼 게임

## 기능 1. 유저 숫자 입력받기
    유저의 숫자를 입력받는 기능을 수행한다.
    세개의 숫자를 입력받아야함.

## 기능 2. 컴퓨터 숫자 입력받기
    컴퓨터는 랜덤으로 숫자 3개를 입력받는다.
    이때, camp.nextstep.edu.missionutils API를 사용한다.

## 기능 3. 결과 출력하기
    같은 수가 같은 자리에 있으면 스트라이크, 다른 자리에 있으면 볼,
    같은 수가 전혀 없으면 낫싱이란 힌트를 얻고, 그 힌트를 이용해서
    먼저 상대방(컴퓨터)의 수를 맞추면 승리한다.
    스트라이크가 3개면 게임 종료

## 기능 4. 게임 재시작
    한 게임이 종료됐을 때,
    1을 입력하면 재시작
    2를 입력하면 프로그램을 종료한다.
