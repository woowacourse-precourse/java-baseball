# 게임 진행
- computer에서 중복 없는 3자리 수가 생성
- user는 해당 난수를 맞추기 위해 3자리의 수를 입력
- 입력받은 수와 난수를 비교
    - 숫자와 자리까지 같다면 strike
    - 숫자만 같다면 ball
    - 같은 수가 없을 경우 낫싱
- 난수와 입력수가 일치할 때 까지 반복
- user가 숫자를 맞출 경우에 재시작/종료 여부를 입력함
    - 1을 입력한 경우 새로운 게임 시작
    - 2를 입력한 경우 게임 종료

---

# 예외 처리
- user에게서 입력 받은 3자리의 수
    - 숫자가 아닌 경우
    - 3자리 숫자가 아닌 경우
    - 중복 되는 수가 있는 경우

- 게임 재시작/종료 여부에 대해 입력 받은 수
    - 1 또는 2가 아닌 다른 수를 입력 받을 경우
