🎯 구현할 기능 목록
=================
### 0. 안내 메시지 출력: 숫자 야구 게임을 시작합니다.
### 1. 랜덤으로 서로 다른 3자리 수 생성
### 2. 사용자로부터 랜덤으로 서로 다른 3자리 수 입력받음
### 3. 입력받은 수에서 볼, 스트라이크 개수를 구해서 반환
* 하나도 없는 경우: 낫싱
* 3개의 숫자를 모두 맞힐 경우: 3스트라이크\n3개의 숫자를 모두 맞히셨습니다! 게임 종료\n게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
* 볼만 있는 경우: n볼
* 스트라이크만 있는 경우: n스트라이크
* 스트라이크, 볼이 모두 있는 경우: n볼 m스트라이크
### 4. 게임이 끝난 경우, 재시작/종료를 구분하는 1과 2 중 하나를 입력받음
* 1을 입력받은 경우: 1번으로 돌아감
* 2를 입력받은 경우: 프로그램 종료
