구현해야할 기능 목록
=
***
1. 임의의 3자리 생성기
2. 3자리 숫자 입력 받기
3. 입력을 사용하기 좋게 조작   
   - 예외 사항 - 입력값이 3자리가 아닐경우
4. 숫자야구 메서드
   - 입력값과 정답값과의 비교
   - 비교값을 리스트형식 "[스트라이크, 볼]" 의 자료로 리턴 
5. 3번의 리턴값에 따른 출력 메서드
   - [0, 0] 일 경우 낫싱 출력
   - [스트라이크(int), 볼(int)] 값에 따라 출력
   - [3, 0] 일 경우 3스트라이크 출력
6. 새로 게임을 시작하거나 종료하는 메서드
   - 반복 선택시 다시 2번으로
   - 종료 선택시 종료문구
