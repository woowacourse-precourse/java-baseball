## 기능 구현

1. 게임 진행
    - 게임 시작, 재시작, 종료 기능
    - 랜덤으로 지정된 수의 기본 세팅 기능
    - 볼, 스트라이크를 판단하는 기능
        - 볼의 개수 세는 기능
        - 스트라이크의 개수 세는 기능
2. 입출력
    - 랜덤 값을 지정하는 기능
    - 유저에게 3자리 수를 입력받는 기능
    - 유저에게 게임 종료, 재시작 (1, 2)를 입력받는 기능
    - 게임을 진행하는 메시지를 출력하는 기능
    - 볼, 스트라이크를 출력하는 기능
    - 게임 종료시 재시작 여부를 출력하는 기능
3. 예외 처리
    - 입력된 3자리 수의 값이 유효한지 체크하는 기능
    - 입력된 재시작, 종료 (1, 2)의 값이 유효한지 체크하는 기능
