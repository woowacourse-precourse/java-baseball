# 구현할 기능 목록

- 게임 시작
- 랜덤 숫자 생성
    - 세 자리 자연수
    - 각 숫자가 서로 달라야 함
    - 1~9로만 이루어짐
- 사용자로부터 값 입력받기
- 입력받은 숫자에 대한 유효성 검사
    - 유효하지 않은 경우, `IllegalArgumentException` 발생
        - 숫자가 아닌 문자를 입력한 경우
        - 세 자리를 입력하지 않은 경우
        - 중복된 수가 있는 경우
        - 0이 존재하는 경우
    - 유효한 경우, 다음 단계 진행
- 볼 및 스트라이크 개수 세기
    - 숫자와 위치가 모두 동일한 경우 스트라이크
    - 숫자만 같고 위치가 다른 경우 볼
    - 숫자가 다른 경우 낫싱
- 결과 출력
    - 3스트라이크: 게임 종료
    - 그 외: `사용자로부터 값 입력받기` 파트부터 반복
- 게임 재진행 여부 묻기
- 입력받은 답변에 대한 유효성 검사
    - 1: 게임 재시작
    - 2: 게임 종료
    - 그 외: `IllegalArgumentException` 발생
- 게임 종료
