## 구현할 기능 목록

### 컴퓨터 랜덤값 생성
> 1. 랜덤값 테스트케이스 작성  
> 2. 게임 랜덤값 생성 (서로 다른 3자리 1~9)

### 플레이어 입력
> 3. 플레이어 입력값 테스트케이스 작성  
> 4. 플레이어 숫자 입력 (서로 다른 3자리 1~9)
> 5. 플레이어 예외 입력 시 프로그램 종료

### 볼 카운트 체크
> 6. 스트라이크 볼 카운트 테스트케이스 작성  
> 7. Strike 판단  
> 8. Ball 판단  
> 9. Nothing 판단
> 10. 3 Strike 시 게임 종료

### 게임 종료, 재시작?
> 11. 게임 재시작 or 프로그램 종료
