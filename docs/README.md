
### 기능을 구현할 순서 

1. 랜덤한 숫자를 뽑아 저장하는 함수

- 입력값의 뒷자리(1의 자리 숫자)부터 검사.
2. 확인하려는 입력값의 숫자가 스트리이크인지 확인한다. 
3. 확인하려는 입력값의 숫자가 볼인지 확인한다. 

- 입력값에 대한 에러 처리
4. 입력한 숫자가 세자리 수인지 확인
5. 입력한 숫자가 1, 2중 하나가 아닌지 확인

### 에러 상황o
입력시
1. 입력 숫자가 3자리 숫자가 아닐 때

게임 프로그램 안에서
1. 만들어진 랜덤 숫자에 동일한 숫자가 있을 때
   -> 다시 랜덤한 숫자를 뽑는다.
