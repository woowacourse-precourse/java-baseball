## 🎮 게임 진행

- [x] 서로 다른 임의의 수 3개 선택하여 난수 생성
- [x] 플레이어로부터 서로 다른 3개의 숫자 입력
- [x] 입력한 수와 생성한 난수 비교
- [x] 같은 수 같은 자리 strike, 다른 자리 ball, 그 외 낫싱 출력
- [x] 3개 숫자를 모두 맞히면 게임 종료 후 재시작/종료 여부 입력
- [x] 1은 재시작, 2는 게임 종료

## 🎲 예외 및 고려사항

- [x] 입력 받은 문자열이 유효한 세 자리의 중복 없는 숫자인지
- [x] 게임 종료 시 1/2 외의 다른 값이 들어왔는지
- [x] depth는 최대 2
